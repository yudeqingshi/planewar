# 高级语言程序设计大作业实验报告

## 一、作业题目

基于 C++ 的飞机大战小游戏设计与实现

---

## 二、开发软件

- 开发语言：C++
- 开发环境：Visual Studio
- 图形化平台：Visual Studio C++ 图形化窗口程序
- 代码托管平台：GitHub
- 项目名称：Plane-War

---

## 三、课题要求

本项目按照高级语言程序设计大作业要求，使用 C++ 语言完成一个图形化小程序。

项目主要实现一个飞机大战小游戏，玩家通过控制我方飞机移动并发射子弹，击毁敌机获得分数。程序包含图形界面、键盘交互、敌机生成、子弹发射、碰撞检测、计分系统和排行榜保存等功能。

---

## 四、主要流程

### 1. 整体流程

程序运行后，首先完成游戏窗口初始化和图片资源加载，包括背景图片、玩家飞机图片、敌机图片和子弹图片。

游戏开始后，程序进入主循环。在主循环中不断完成以下操作：

1. 绘制游戏背景和游戏对象；
2. 检测玩家键盘输入；
3. 更新玩家飞机位置；
4. 生成并移动子弹；
5. 生成并移动敌机；
6. 判断子弹和敌机是否碰撞；
7. 更新分数；
8. 判断游戏是否结束；
9. 游戏结束后保存分数到排行榜文件。

整体流程如下：

```text
程序启动
   ↓
初始化窗口和图片资源
   ↓
进入游戏主循环
   ↓
键盘输入检测
   ↓
飞机移动、子弹移动、敌机移动
   ↓
碰撞检测
   ↓
分数更新
   ↓
判断游戏是否结束
   ↓
保存排行榜
